# MakeupGame — Система нанесения макияжа

![Unity Version](https://img.shields.io/badge/Unity-6.3%20LTS-black?logo=unity)
![C#](https://img.shields.io/badge/C%23-9.0-blue?logo=c-sharp)

Проект представляет собой реализацию механики нанесения макияжа на куклу в среде Unity. Разработан в рамках тестового задания. Основная цель — демонстрация навыков работы с архитектурой кода, скриптуемыми объектами и анимациями.

## 🎮 Особенности проекта

- **Разнообразие инструментов:** Реализованы инструменты для разных зон лица.
- **Система смывания:** Возможность удалить макияж с помощью спонжа.
- **Гибкая настройка:** Все параметры инструментов вынесены в `ScriptableObject`.
- **Плавные анимации:** Использование библиотеки **DoTween** для визуальной обратной связи.

### Типы макияжа
1. **Тени для век** (Eyeshadows)
2. **Румяна** (Blush)
3. **Помада** (Lipstick)
4. **Тональный крем** (Foundation)
5. **Спонж** (Инструмент для удаления)

## 🛠 Технический стек

- **Engine:** Unity 6.3 LTS
- **Language:** C#
- **Animations:** DoTween
- **Architecture:** OOP (Наследование), ScriptableObject

## 🏗 Архитектура кода

Проект построен на основе объектно-ориентированного подхода с использованием паттерна наследования для инструментов.

- **`BaseTool`**: Базовый класс, содержащий общую логику взаимодействия, ввода и конфигурации инструмента.
- **Наследники**:
  - `Cream` (Тональный крем)
  - `BlushBrush` (Кисть для румян)
  - `ShadowsBrush` (Кисть для теней)
  - `Pomade` (Помада)
- **Конфигурация**: Каждый инструмент имеет свой ассет `ScriptableObject`, что позволяет настраивать силу нанесения, цвет, размер кисти и другие параметры без изменения кода.
